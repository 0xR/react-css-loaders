# React CSS Loaders

A collection of pure CSS React loading components. Based on Luke Haas [css-loaders](https://github.com/lukehaas/css-loaders) project.

## Getting Start

```bash
npm install react-css-loaders --save
```

## Usage

``` javascript
  import { BarLoader } from 'react-css-loaders';
  ...
  <BarLoader />
  ...
```
